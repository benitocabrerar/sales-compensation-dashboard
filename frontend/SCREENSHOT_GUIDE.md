## Screenshot Specifications

### For Best Quality

#### Resolution Settings
```
Desktop: 1920×1080 (Full HD)
Tablet:  1024×768
Mobile:  375×667 (iPhone SE)
```

#### Browser Settings
- **Zoom:** 100% (no zoom)
- **Window:** Maximized
- **DevTools:** Closed
- **Extensions:** Disabled (if possible)

#### Capture Tools
**Windows:**
- Windows + Shift + S (Snipping Tool)
- Print Screen + Paint
- Snagit (professional)

**Mac:**
- Command + Shift + 4 (area)
- Command + Shift + 3 (full screen)

**Browser Extensions:**
- Awesome Screenshot
- Nimbus Screenshot
- Full Page Screen Capture

## Video Recording Guide

### For Demo/Tutorial

#### Recommended Flow

**Part 1: Overview (30 seconds)**
1. Show full dashboard homepage
2. Highlight company name
3. Pan across all three tabs
4. Show footer with version

**Part 2: Sales Dashboard (2 minutes)**
1. Click Sales Dashboard tab
2. Scroll from top to bottom slowly
3. Hover over metric cards (show animation)
4. Change input values (show live calculation)
5. Scroll through economics sections
6. Highlight key numbers

**Part 3: JobNimbus Analytics (2 minutes)**
1. Click JobNimbus Analytics tab
2. Show hero section stats
3. Hover over monthly chart (show tooltips)
4. Click pie chart legend items
5. Scroll to revenue timeline
6. Show insight cards

**Part 4: P&L Analysis (2 minutes)**
1. Click P&L Analysis tab
2. Show financial metrics
3. Compare best vs worst cards
4. Hover over trend chart (show tooltips)
5. Show COGS pie chart
6. Review OpEx bar chart
7. Read insight cards

**Part 5: Interactivity Demo (1 minute)**
1. Switch between tabs quickly
2. Show hover animations
3. Demonstrate chart tooltips
4. Test input fields
5. Show responsive behavior (optional)

#### Recording Settings
```
Resolution: 1920×1080
Frame Rate: 60 fps
Format: MP4
Codec: H.264
Audio: Optional (voice-over)
Length: 5-7 minutes total
```

#### Tools
- **OBS Studio** (Free, professional)
- **Loom** (Easy, cloud-based)
- **Camtasia** (Professional, editing)
- **ScreenFlow** (Mac, professional)

## Screenshot Checklist

### Before Capturing

- [ ] Dashboard is fully loaded
- [ ] All data is displaying correctly
- [ ] Charts are rendered completely
- [ ] Browser is at 100% zoom
- [ ] Window is appropriate size
- [ ] No personal information visible
- [ ] No browser notifications
- [ ] Clean, professional appearance

### After Capturing

- [ ] Image is clear and sharp
- [ ] Colors are accurate
- [ ] Text is readable
- [ ] File is properly named
- [ ] Resolution is adequate
- [ ] Format is appropriate
- [ ] File size is reasonable
- [ ] Saved in correct location
